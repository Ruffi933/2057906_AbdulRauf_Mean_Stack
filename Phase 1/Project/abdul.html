Types of Man in the Middle Attack


  • 1) ARP Spoofing

    • In an ARP Spoofing, the goal of the attacker is to capture packets, inject his own information and potentially alter the information being shared between the client and the server.


  • 2) DNS Spoofing

    • In this type of attack, the attacker injects false IP addresses into the DNS cache which results in landing the user to malicious web pages.
